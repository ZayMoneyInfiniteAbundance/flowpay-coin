# Security Policy

## Reporting a Vulnerability

Please report security issues directly to the maintainer via GitHub Issues (private if possible) or email `security@flowpay-coin.org` (placeholder).

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |
